# Plan de Trabajo: Crear Web con Cloud de Codes + Claude Code

## Instrucciones paso a paso

### Paso 1: Crear carpeta en escritorio
- Crear una carpeta en el escritorio
- Nombre sugerido: `creacion-web` (o el nombre que prefieras)
- Esta carpeta albergará todo el proyecto: imágenes, skills, instrucciones y archivos

### Paso 2: Descargar 3 skills desde Cloud de Codes Templates
1. **Señor Backend** - Para el backend (lo que no se ve: bases de datos, formularios, funcionalidad de botones)
2. **Señor Frontend** - Para el frontend (parte visual, lo que el usuario ve y toca)
3. **Sea Optimizer** - Para optimizar textos y mejorar ventas/SEO

### Paso 3: Añadir skills al proyecto en Cloud de Codes
- Abrir Cloud de Codes
- Pedir que añada las tres skills a la carpeta del proyecto creada en el Paso 1
- Cloud de Codes detectará y aplicará las skills automáticamente

### Paso 4: Crear documento con Cloud de Codes
- Crear un documento con los datos más importantes para la web
- Una vez creado, añadir el documento a la carpeta del proyecto
- Dentro de esta carpeta también se añaden vídeos e imágenes a utilizar

### Paso 5: Construir la web
- Decirle a Cloud de Codes que construya la web
- Esperar el resultado
