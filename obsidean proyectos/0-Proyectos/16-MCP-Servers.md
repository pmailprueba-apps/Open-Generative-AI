## Visión General

Servidores MCP (Model Context Protocol) para conectar LLMs con herramientas y servicios externos.

---

## ¿Qué es MCP?

Model Context Protocol es un protocolo que permite a modelos de IA conectarse con herramientas externas, bases de datos, APIs, y otros servicios de manera estandarizada.

---

## Uso

Los servidores MCP permiten:
- Extender capacidades de Claude Code
- Conectar con bases de datos
- Integrar con APIs externas
- Automatizar tareas de desarrollo
