## Visión General

Herramienta diseñada para la conversión dinámica de código HTML a imágenes PNG/JPG y documentos PDF de alta fidelidad. Es ideal para automatizar reportes, generar facturas o capturar layouts de páginas web de forma programática.

---

## Características principales

- Renderizado de layouts complejos.
- Exportación a PDF vectorizado y optimizado.
- Generación de capturas en múltiples dimensiones (desktop, mobile).
- Procesamiento batch.

---

## Tecnologías

- JavaScript / Node.js
- Puppeteer / Playwright para control headless del navegador.
- HTML5, CSS3 modernos (flexbox, grid).
