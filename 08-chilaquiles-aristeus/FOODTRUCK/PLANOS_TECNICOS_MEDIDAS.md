# Planos Técnicos con Medidas Estimadas: Foodtruck Aristeus 📏📐

Basado en las dimensiones estándar de un camión de servicio mediano (Step Van) y las proporciones de las fotos proporcionadas, aquí tienes el desglose técnico de medidas para la remodelación.

---

## 1. Vista Lateral (Costado de Servicio)
Es el área más importante para la operación y ventas.

| Elemento | Medida Sugerida | Notas |
| :--- | :--- | :--- |
| **Longitud Total** | 5.50 m | Dimensión externa aproximada del vehículo. |
| **Altura Total** | 2.80 m | Desde el suelo hasta el techo (sin incluir aire acondicionado). |
| **Ventanilla de Servicio** | 2.00 m x 0.90 m | Centrada para atención a dos clientes simultáneos. |
| **Altura de Mostrador** | 1.10 m | Altura ergonómica desde el suelo exterior. |
| **Puerta de Acceso** | 0.80 m x 1.90 m | Generalmente situada en el lado del copiloto o trasera. |

## 2. Vista Frontal (Frente)
Área de la cabina y branding principal.

| Elemento | Medida Sugerida | Notas |
| :--- | :--- | :--- |
| **Ancho Total** | 2.20 m | Ancho estándar para circulación urbana. |
| **Área de Letrero Superior** | 2.00 m x 0.40 m | Espacio ideal para el logo "Aristeus" retroiluminado. |
| **Altura Libre (Interior)** | 2.10 m | Espacio de cabeza para el personal. |

## 3. Vista Trasera (Posterior)
Zona de carga y servicios técnicos.

| Elemento | Medida Sugerida | Notas |
| :--- | :--- | :--- |
| **Puerta Trasera** | 1.00 m x 1.80 m | Acceso para carga de insumos pesados. |
| **Compartimento Gas** | 0.60 m x 0.80 m | Espacio exterior para 2 tanques de 10kg/20kg. |
| **Luces de Seguridad** | 0.20 m | Diámetro de luces LED de alto brillo. |

## 5. Recomendaciones de Materiales
*   **Estructura:** Refuerzos de acero galvanizado para equipos pesados.
*   **Recubrimiento:** Paneles de aluminio de 3mm (ACM) para el exterior.
*   **Aislamiento:** Lana de roca o poliestireno extruido para control de calor en la zona de cocina.

---
**IMPORTANTE:** Estas medidas son estimaciones basadas en modelos estándar. Se recomienda realizar una **medición física con cinta métrica** sobre el vehículo real antes de comprar equipos empotrados o mandar a imprimir vinilos de gran formato.
