# 🌶️ Proyecto: Menú Interactivo Aristëus (Next Gen)

## 🎯 Objetivo
Transformar el menú estático en una aplicación web interactiva que permita a los clientes personalizar su orden y enviarla directamente por WhatsApp con un solo clic.

## 🎨 ADN Visual (Industrial Elegance)
- **Colores:** Fondo Negro Carbon (#121212), Detalles en Oro/Ámbar (#FFD700), Acentos en Rojo Chile (#B22222).
- **Efectos:** Glassmorphism (efecto de vidrio esmerilado) en las tarjetas de selección.
- **Tipografía:** Inter (Moderna) y Outfit (Elegante).

## 🛠️ Stack Técnico Propuesto
- **Framework:** Vite + Vanilla JS (Máximo rendimiento).
- **Estilos:** CSS Vanilla con variables personalizadas.
- **Lógica:** State management simple para carrito de compras.

## 🚀 Hoja de Ruta (Pasos para cuando se decida implementar)
1. **Fase 1: Estructura Móvil** - Crear el layout tipo "App Nativa".
2. **Fase 2: Selector de Ingredientes** - Implementar el flujo de: Salsa -> Proteína -> Extras.
3. **Fase 3: Motor de Costos** - Cálculo dinámico de precios.
4. **Fase 4: Puente WhatsApp** - Generación automática del mensaje estructurado para cocina.
