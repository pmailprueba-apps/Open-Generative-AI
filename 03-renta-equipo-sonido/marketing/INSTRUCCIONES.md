## FASE 1: FUNDACIÓN (Semanas 1-2)

### Semana 1 - Investigación y Posicionamiento

#### Día 1-2: Research
```bash
# Usar skills:
customer-research + competitor-profiling + pricing-strategy
```

**Prompts a usar:**
1. "Investiga el mercado de renta de equipo de sonido en San Luis Potosí. Qué tipos de eventos predominan (bodas, quinceaños, corporativos)? Qué presupuesto manejan? Qué buscan: precio, calidad, ambos?"

2. "Analiza a los 10 competidores principales de renta de sonido en SLP. Compara: equipos, precios, cobertura, reputación. Identifica gaps."

3. "Investiga estructura de precios de la industria: cómo rentan sonido otros proveedores? Por hora? Por evento? Paquetes? Incluye rentals de equipos de gama alta vs económica."

#### Día 3-4: Posicionamiento Premium
```bash
# Usar skills:
brand-guidelines + copywriting + site-architecture
```

**Prompts a usar:**
1. "Crea brand-guidelines para empresa de renta de sonido premium en SLP. Diferenciador: calidad profesional + servicio confiable. Colores: negro, dorado, blanco. Tipografía moderna pero seria."

2. "Desarrolla posicionamiento: NO somos 'el equipo del pueblo'. Somos 'sonido profesional para eventos que importan'. Crea 3 variantes de tagline."

3. "Diseña site-architecture para el sitio web: Home (propuesta de valor), Equipos (catálogo), Servicios (bodas, corporativos, conciertos), Galería, Cotizador, Contacto."

#### Día 5-7: Catálogo de Equipos
```bash
# Usar skills:
social-content + image-enhancer
```

**Prompts a usar:**
1. "Crea catálogo de equipos de sonido: beschreibung de cada pieza (bocinas, mezcladoras, microfonos, luces), capacidades, precios por hora/día. Para web y PDF."

2. "Genera fotos profesionales de equipos de sonido: fondos oscuros, iluminación dramática, estilo catálogo premium. Necesitamos 15-20 imágenes de alta calidad."

3. "Prepara content-strategy para Instagram enfocando en: 40% mostrar equipos (cualidad técnica), 30% eventos realizados (social proof), 20% tips de audio (educación), 10% ofertas."

## FASE 2: LANZAMIENTO (Semanas 3-4)

### Semana 3 - Presencia Digital

```bash
# Usar skills:
seo-audit + copywriting + schema-markup
```

**Prompts a usar:**
1. "Ejecuta seo-audit para sitio de renta de sonido. Optimizar para: 'renta de sonido San Luis Potosi', 'equipo de sonido para bodas SLP', 'sonido profesional para eventos'."

2. "Escribe copy persuasivo para cada página del sitio. Cada sección debe responder: qué hacemos, por qué somos diferentes, cómo contratar."

3. "Implementa schema-markup: LocalBusiness, Service, Product (para equipos en venta si aplica). Buscar rich snippets en Google."

### Semana 4 - Captura de Leads

```bash
# Usar skills:
page-cro + form-cro + cold-email
```

**Prompts a usar:**
1. "Optimiza page-cro de la página de contacto/cotizador. Objetivo: que cada visitante cotize o llame. Diseño limpio, CTAs claros, WhatsApp prominente."

2. "Crea form-cro optimizado para cotización: campos esenciales (nombre, tipo de evento, fecha, ubicación, # de asistentes), resto en seguimiento posterior."

3. "Ejecuta cold-email campaign a salones de eventos y wedding planners de SLP: 50 emails personalizados presentando el servicio."

## FASE 3: CRECIMIENTO (Semanas 5-8)

### Semana 5 - Publicidad

```bash
# skills:
paid-ads + ad-creative + video
```

**Prompts a usar:**
1. "Crea estrategia de paid-ads para Google Ads: $4,000/mes. Keywords: 'renta de sonido [ciudad]', 'equipo para boda [ciudad]', 'sonido profesional eventos'. Landing pages dedicadas."

2. "Genera 8 creatividades para Facebook/Instagram Ads: antes/después de eventos, videos cortos de equipos en acción (reels), testimonios de clientes."

3. "Crea video-reel de un evento completo: setup del equipo, el evento en progreso, breakdown. 60 segundos para Instagram Reels."

### Semana 6 - Alianzas

```bash
# skills:
sales-enablement + partnership + referral-program
```

**Prompts a usar:**
1. "Crea sales-enablement kit para_presentaciones B2B: презентация de 10 slides con equipos, casos de éxito, precios, diferenciadores. Para presentar a empresas y salones."

2. "Desarrolla programa de parceria com salones de eventos: comisión por cada referral que contrate, descuento para sus clientes."

3. "Implementa referral-program: clientes que refieran a otro cliente que contrate, ganan descuento en siguiente rental."

### Semana 7-8 - Optimización

```bash
# skills:
analytics-tracking + ab-test-setup + churn-prevention
```

**Prompts a usar:**
1. "Implementa analytics-tracking completo: Google Analytics, Facebook Pixel, WhatsApp Business analytics. Dashboard semanal de métricas."

2. "Configura ab-test-setup para site: test de headlines, test de formulario corto vs largo, test de CTA de WhatsApp vs formulario web."

3. "Crea estrategia de churn-prevention para clientes corporativos: check-ins post-evento, mantenimiento preventivo de equipos, offers de renovación."
