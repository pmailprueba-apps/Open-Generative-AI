# 📸 Shot List: Italika FT150 — Review de 1 Año

Este documento contiene la lista de tomas (shot list) y el equipo mínimo necesario para la grabación en campo del primer video del canal.

## 🛠️ Equipo Mínimo Requerido

- [ ] **Cámara:** Smartphone (resolución mínima 1080p a 30/60 FPS).
- [ ] **Audio:** Micrófono de solapa (Lavalier) con adaptador para smartphone/cámara.
- [ ] **Estabilidad:** Trípode para celular o estabilizador (gimbal) básico.
- [ ] **Accesorios:** Soporte de manillar para tomas POV (tipo GoPro o adaptador para celular).

## 💡 Consejos prácticos para la grabación
1. **Audio:** Haz pruebas de sonido antes de empezar a grabar. Monitorea que el viento no golpee directamente el micrófono usando un protector de espuma (deadcat).
2. **Iluminación:** Graba durante la "hora dorada" (cerca del atardecer) para la escena del outro y las tomas de B-roll estéticas.
3. **Cámara lenta:** Usa el modo slow motion (120 FPS o 240 FPS si tu celular lo permite) exclusivamente para los planos de frenado o paso rápido.
