# 🍳 PROYECTO 07 — COOKING EXPRESS

## Objetivo
Generar imágenes de comida a partir de un logotipo existente usando IA generativa (Flux, DALL-E, Midjourney, etc.) para crear contenido visual de marketing para restaurantes, servicios de catering, o marcas de comida.

---

## FASE 1 — FUNDACION (Semana 1)

### Dia 1-2: Analisis del Logotipo
- Recepcions y analisis del logotipo existente (formato, colores, estilo)
- Definir estilo visual objetivo (comida casera, gourmet, rápida, ethnic)
- Identificar industrias objetivo (restaurantes, food trucks, servicios de catering, marcas de snacks)
- customer-research: analizar competencia que ofrece servicios similares

### Dia 3-4: Branding y Estilo Visual
- brand-guidelines: crear guía de estilo para el proyecto
- Definir paleta de colores complementarios para las imágenes generadas
- Crear prompt templates estándar por tipo de cocina
- Definir formatos de salida (redes sociales, ads, menú, etc.)

### Dia 5: Configuracion de Herramientas
- Configurar acceso a herramientas de generación de imágenes (Flux, DALL-E, etc.)
- Preparar workflow de generación con el logotipo como base
- Definir proceso de composición (logotipo + imagen de comida)
- Crear biblioteca de prompts reutilizables

### Dia 6-7: Entregables Fase 1
- [ ] Análisis del logotipo existente
- [ ] Guía de estilo visual del proyecto
- [ ] Biblioteca de prompts por tipo de cocina
- [ ] Workflow de generación documentado
- [ ] 5-10 imágenes de muestra generadas

---

## FASE 2 — LANZAMIENTO (Semana 2)

### Dia 8-9: Creación de Portafolio Inicial
- Generar imágenes para 5 categorías de comida distintas
- Crear variaciones por cada categoría (diferentes estilos, ángulos, formatos)
- Compilar imágenes en catálogo digital

### Dia 10-11: Estrategia Comercial
- Definir modelo de negocio (venta unitaria, suscripción, paquete)
- Definir precios por tipo de cliente
- sales-enablement: crear presentación comercial
- cold-email: crear secuencia para restaurantes y food trucks

### Dia 12-13: Marketing y Captación
- social-content: calendario de contenido para mostrar ejemplos
- paid-ads: crear anuncios mostrando antes (sin imagem) / después (con imagem generada)
- Definir lead-magnet: "10 imágenes gratis de comida para tu restaurante"

### Dia 14: Entregables Fase 2
- [ ] Catálogo de 30-50 imágenes de muestra
- [ ] Lista de precios
- [ ] Presentación comercial
- [ ] Campaña de ads inicial

---

## FASE 3 — CRECIMIENTO (Semanas 3-4)

### Semana 3: Escalamiento
- Optimizar workflow de generación (automatizar con scripts si aplica)
- Crear paquete de "sesión de fotos virtual" para restaurantes
- mcp-builder: automatizar prompts más comunes
- Referral-program: programa "Recomienda y obtén imágenes gratis"

### Semana 4: Automatización y Retención
- Analytics-tracking: métricas de ventas y uso
- churn-prevention: follow-up con clientes B2B
- pricing-strategy: descuentos por volumen o suscripción mensual
- video: crear tutorial de cómo usar el servicio
